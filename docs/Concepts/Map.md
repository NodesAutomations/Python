### Map
```python
def cube(x):
    return x*x*x

numbers = [1, 2, 3, 4]
cubeList = list(map(cube, numbers))
print(cubeList)
```
### Filter

```python
def IsEven(x):
    return (x % 2) == 0

numbers = [1, 2, 3, 4]
filteredList = list(filter(IsEven, numbers))
print(filteredList)
```
