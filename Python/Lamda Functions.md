### Overview
- Lamda function is a small anonymous functions without name

### Sample code
```python
square=lambda x:x*2
cube=lambda x:x*3

print(square(5)) # Prints 25
print(cube(5)) # Prints 125
```
### function with multiple varriables
```python
avg=lambda x,y,z:(x+y+z)/3

print(avg(1,2,3)) # print 2
```
